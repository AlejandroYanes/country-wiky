Countries API Documentation

This API provides country data with various filtering options.

Endpoints

Get All Countries

Returns a list of all countries.

GET /countries

Get Country by Name

Searches for a country by partial name.

GET /countries/name/{name}

Get Country by Full Name

Searches for a country by exact full name.

GET /countries/fullname/{name}

Get Country by Code

Searches for a country by its cca2, cca3, ccn3, or cioc code.

GET /countries/code/{code}

Get Countries by Multiple Codes

Retrieve multiple countries by passing a list of codes as a query parameter.

GET /countries/codes?codes=USA,CAN,MEX

Filtering Fields

You can filter the response to include only specific fields using the fields query parameter.

GET /countries?fields=name.common,capital
