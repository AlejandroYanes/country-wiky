Countries API

A powerful API to access country data.

Getting Started

Base URL: http://localhost:8080

Use the following endpoints to retrieve country data.

Endpoints

Get All Countries

Returns a list of all countries.

GET /countries

Get Country by Name

Search for a country by name.

GET /countries/name/{name}

Get Country by Full Name

Search for a country by exact full name.

GET /countries/fullname/{name}

Get Country by Code

Search by cca2, cca3, ccn3, or cioc country code.

GET /countries/code/{code}

Get Countries by Multiple Codes

Retrieve multiple countries using a comma-separated list.

GET /countries/codes?codes=USA,CAN,MEX

Get Country by Currency

Search for countries using a currency code.

GET /countries/currency/{currency}

Get Country by Language

Search for countries by language.

GET /countries/lang/{language}

Get Country by Region

Retrieve countries by region.

GET /countries/region/{region}

Filtering Fields

You can filter the response to include only specific fields using the fields query parameter.

GET /countries?fields=name.common,capital
